Add-ons axés sur la protection de la vie privée Privacy Oriented Browser Add-ons :

  • CanvasBlocker (Source) : Bloque le Fingerprinting via Canvas.
  • Chameleon (Source) : User Agent Spoofer.
  • Cookie AutoDelete (Source) : Nettoyeur de Cookies avec des règles personnalisées.
  • Decentraleyes (Source) : Utilise des bibliothèques intégrées vous évitez les CDN.
  • HTTPS Everywhere (Source) : Force l'utilisation de HTTPS quand cela est possible (ou en permanence).
  • Invidition (Source) : Redirige les liens et embeds YouTube vers une Instance Invidious.
  • Neat URL (Source) : Suprime les Paramètres UTM.
  • Skip Redirect (Source) : Saute les URLs servant au pistage.
  • uBlock Origin (Source) : Bloqueur de pubs (ce n'est PAS "uBlock Plus", "uBlock Mobile", "µBlock").
  • uMatrix (Source) : Point & click firewall.
  • Obtional : Block Cloudflare MiTM Attack (Source + Detail) : Bloque tout les sites utilisant le MITM de Cloudflare.

  • Violentmonkey (Source).

  • Userstyles :
