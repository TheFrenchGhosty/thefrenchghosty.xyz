Indenting/Formating Rules :

- 2 Spaces
- 1 Blank line between : <html> and <head>
- 1 Blank line between : </head> and <body>
- 1 Blank line between : </body> </html>
- 1 Blank line at the end of the file


Example :

```html
<!DOCTYPE html>
<html>

<head>
  <title>A Title</title>
</head>

<body>
  <h1>A Header</h1>
</body>

</html>

```
